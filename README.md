# asset-phonebook
A listing of Online Asset Libraries for Blender

[http://johnnygizmo.github.io/blender-asset-phonebook]

This is a collection of asset libraries you can add to Blender. To submit a addition, create a PR, or send an email to johnny.matthews+phonebook@gmail.com with:

- Your Name
- Email Address
- Contact Website (optional)
- Youtube Channel (optional)
- Asset URL
- Library Website (optional promotional website for library, required if Freemium or Premium)
- Description of the general types of assets in your repo
- Whether your library is 100% free, freemium, or premium only  (if freemium or premium, please include a library website)
